# Use Case 009: 다크 모드 전환 (Toggle Dark Mode)

## 1. 개요

| 항목 | 내용 |
|------|------|
| **유스 케이스 ID** | UC-009 |
| **유스 케이스명** | 다크 모드 전환 |
| **작성일** | 2026-01-10 |
| **버전** | 1.0 |
| **상태** | 초안 |

## 2. 설명

사용자가 라이트 모드와 다크 모드 사이를 전환할 수 있는 기능입니다. 시스템 설정에 따라 자동으로 다크 모드를 적용할 수도 있습니다.

## 3. 액터

| 액터 유형 | 설명 |
|-----------|------|
| **주 액터** | 일반 사용자 |

## 4. 사전 조건

| 조건 | 설명 |
|------|------|
| PRE-01 | 애플리케이션이 실행 중이어야 함 |

## 5. 사후 조건

| 조건 | 설명 |
|------|------|
| POST-01 | 테마 설정이 변경됨 (light ↔ dark) |
| POST-02 | 모든 UI 컴포넌트가 새 테마에 맞게 업데이트됨 |
| POST-03 | 테마 설정이 로컬 스토리지에 저장됨 |

## 6. 기본 흐름

| 단계 | 액터 | 시스템 |
|------|------|--------|
| 1 | 헤더의 다크/라이트 모드 토글 버튼을 클릭함 | |
| | | 2. 테마를 반대로 변경함 |
| | | 3. html 요소에 dark 클래스 토글 |
| | | 4. 로컬 스토리지에 테마 설정 저장 |

## 7. UI 요구사항

### 토글 버튼

**라이트 모드:** 🌙 (다크 모드로 전환)
**다크 모드:** ☀️ (라이트 모드로 전환)

### 테마별 색상 패릿

| 요소 | 라이트 모드 | 다크 모드 |
|------|-------------|-----------|
| 배경 | bg-white | bg-gray-900 |
| 텍스트 | text-gray-900 | text-white |
| 카드 배경 | bg-white | bg-gray-800 |

## 8. 관련 유스 케이스

| 관련 유스 케이스 | 관계 유형 |
|-----------------|-----------|
| 모든 유스 케이스 | 포함 (모든 UI에 영향) |
