# Use Case 007: 우선순위 설정 (Set Priority)

## 1. 개요

| 항목 | 내용 |
|------|------|
| **유스 케이스 ID** | UC-007 |
| **유스 케이스명** | 우선순위 설정 |
| **작성일** | 2026-01-10 |
| **버전** | 1.0 |
| **상태** | 초안 |

## 2. 설명

사용자가 작업의 우선순위를 설정하는 기능입니다. 우선순위는 4단계(높음, 중간, 낮음, 없음)로 구분되며, 작업 카드에 시각적으로 표시됩니다.

## 3. 액터

| 액터 유형 | 설명 |
|-----------|------|
| **주 액터** | 일반 사용자 |

## 4. 사전 조건

| 조건 | 설명 |
|------|------|
| PRE-01 | 작업이 존재해야 함 |
| PRE-02 | 작업 추가/수정 대화상자가 열려 있어야 함 |

## 5. 사후 조건

| 조건 | 설명 |
|------|------|
| POST-01 | 작업에 priority 필드가 설정됨 |
| POST-02 | 작업 카드에 우선순위 아이콘이 색상으로 표시됨 |
| POST-03 | 로컬 스토리지에 변경사항이 저장됨 |

## 6. UI 요구사항

### 우선순위별 색상

| 우선순위 | 색상 | 아이콘 색상 |
|----------|------|-------------|
| 높음 (high) | ⭐ | 빨간색 (text-red-500) |
| 중간 (medium) | ⭐ | 노란색 (text-yellow-500) |
| 낮음 (low) | ⭐ | 초록색 (text-green-500) |
| 없음 (none) | ⭐ | 회색 (text-gray-400) |

### 작업 카드 표시

```
┌─────────────────────────────────────┐
│ ○ │ 작업 제목                   ⭐    │ ← 우선순위 아이콘
│                     [✏️] [🗑️]       │
└─────────────────────────────────────┘
```

## 7. 관련 유스 케이스

| 관련 유스 케이스 | 관계 유형 |
|-----------------|-----------|

| UC-002: 작업 수정 | 포함 (우선순위 수정 포함) |
