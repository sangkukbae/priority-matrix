# Use Case 006: 마감일 설정 및 확인 (Set and View Due Date)

## 1. 개요

| 항목 | 내용 |
|------|------|
| **유스 케이스 ID** | UC-006 |
| **유스 케이스명** | 마감일 설정 및 확인 |
| **작성일** | 2026-01-10 |
| **버전** | 1.0 |
| **상태** | 초안 |

## 2. 설명

사용자가 작업에 마감일을 설정하거나, 설정된 마감일을 확인하는 기능입니다. 상대적 날짜 표기법(오늘, 내일, 3일 후 등)도 함께 표시합니다.

## 3. 액터

| 액터 유형 | 설명 |
|-----------|------|
| **주 액터** | 일반 사용자 |
| **보조 액터** | 시스템 (date-fns 라이브러리) |

## 4. 사전 조건

| 조건 | 설명 |
|------|------|
| PRE-01 | 작업이 존재해야 함 |
| PRE-02 | 작업 추가/수정 대화상자가 열려 있어야 함 |

## 5. 사후 조건

| 조건 | 설명 |
|------|------|
| POST-01 | 작업에 dueDate 필드가 설정됨 |
| POST-02 | 작업 카드에 마감일이 표시됨 |
| POST-03 | 로컬 스토리지에 변경사항이 저장됨 |

## 6. 기본 흐름

### 6.1 마감일 설정 (대화상자에서)

| 단계 | 액터 | 시스템 |
|------|------|--------|
| 1 | "마감일" 필드를 클릭함 | |
| | | 2. 날짜 선택 캘린더가 표시됨 |
| 3 | 날짜를 선택함 | |
| | | 4. 선택된 날짜가 필드에 표시됨 |

### 6.2 마감일 확인 (작업 카드에서)

| 단계 | 액터 | 시스템 |
|------|------|--------|
| 1 | 작업 카드를 봄 | |
| | | 2. 마감일이 설정된 경우 📅 아이콘과 함께 표시됨 |
| | | 3. 상대적 날짜 계산하여 함께 표시 |

## 7. UI 요구사항

### 상대적 날짜 표시 규칙

| 상대적 날짜 | 조건 |
|-------------|------|
| "오늘" | 마감일 === 오늘 |
| "내일" | 마감일 === 내일 |
| "N일 후" | 2일 ~ 7일 후 |
| "2026년 1월 15일" | 7일 초과 |

### 마감일 상태별 색상

| 상태 | 색상 | 조건 |
|------|------|------|
| 정상 | 회색 | 7일 이상 남음 |
| 경고 | 노란색 | 3일 ~ 6일 남음 |
| 긴급 | 빨간색 | 3일 미만 남음 |
