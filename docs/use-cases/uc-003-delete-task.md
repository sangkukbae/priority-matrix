# Use Case 003: 작업 삭제 (Delete Task)

## 1. 개요

| 항목 | 내용 |
|------|------|
| **유스 케이스 ID** | UC-003 |
| **유스 케이스명** | 작업 삭제 |
| **작성일** | 2026-01-10 |
| **버전** | 1.0 |
| **상태** | 초안 |

## 2. 설명

사용자가 작업을 삭제하는 기능입니다. 삭제 전 확인 대화상자를 통해 실수를 방지합니다.

## 3. 액터

| 액터 유형 | 설명 |
|-----------|------|
| **주 액터** | 일반 사용자 |
| **보조 액터** | 시스템 (로컬 스토리지) |

## 4. 사전 조건

| 조건 | 설명 |
|------|------|
| PRE-01 | 삭제할 작업이 존재해야 함 |
| PRE-02 | 사용자가 삭제할 작업을 선택해야 함 (삭제 버튼 클릭) |

## 5. 사후 조건

| 조건 | 설명 |
|------|------|
| POST-01 | 작업이 목록에서 제거됨 |
| POST-02 | 해당 사분면의 작업 카운트가 1 감소함 |
| POST-03 | 로컬 스토리지에서 작업 데이터가 삭제됨 |
| POST-04 | 삭제 완료 토스트 메시지가 표시됨 |

## 6. 기본 흐름

| 단계 | 액터 | 시스템 |
|------|------|--------|
| 1 | 작업 카드에서 "삭제" 버튼을 클릭함 | |
| | | 2. 확인 대화상자가 표시됨 |
| | | 3. "정말로 삭제하시겠습니까?" 메시지와 함께 작업 제목 표시 |
| 3 | "삭제" 버튼을 클릭함 | |
| | | 4. 해당 Task 객체를 스토어에서 제거함 |
| | | 5. 로컬 스토리지에 데이터를 저장함 |
| | | 6. 작업 카드가 화면에서 사라짐 (애니메이션과 함께) |
| | | 7. 해당 사분면 카운트가 감소함 |
| | | 8. "삭제되었습니다" 토스트 메시지 표시 |

## 7. 대체 흐름

| 대체 흐름 | 조건 | 설명 |
|-----------|------|------|
| ALT-01 | 취소 버튼 클릭 | 삭제 취소, 대화상자 닫힘 |
| ALT-02 | 대화상자 외부 클릭 | 삭제 취소, 대화상자 닫힘 |

## 8. 예외 상황

| 예외 상황 | 처리 방법 |
|-----------|-----------|
| EXC-01 | 작업이 이미 삭제됨 | "작업이 존재하지 않습니다" 메시지 표시 |
| EXC-02 | 로컬 스토리지 오류 | 오류 로깅 후 재시도 옵션 제공 |

## 9. 트리거

| 트리거 유형 | 설명 |
|-------------|------|
| 사용자 클릭 | 작업 카드의 삭제 (🗑️) 버튼 클릭 |
